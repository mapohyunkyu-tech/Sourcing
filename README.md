# MarketScout 시즌 AI v3.9

## 이번 버전 핵심
- 백업 DB에서 병합한 품목을 `item_master.csv`와 새 설치용 기본 DB에 포함
- 품목 검색 화면에 사진 활용 가이드 표시
  - 같은 품종·동일 유통명
  - 겉모양 유사 품종
  - 단면 유사 품종
  - 겉색·과육색
  - 사진 대체 사용 가능 여부와 주의사항
- 기존 설치 DB에도 CSV의 신규 품목과 사진 가이드를 자동 동기화

## 기본 DB 업데이트 기능
`🗂 DB 백업` 탭 아래의 **⚙️ 기본 DB 업데이트**에서 실행합니다.

버튼을 누르면 다음 작업을 자동으로 수행합니다.
1. 현재 사용 중인 DB의 활성 품목과 사진 가이드 추출
2. `item_master.csv`에 병합
3. `3분류_정규화` 기준 중복 제거
4. 분석 원자료·API 로그·대기열이 비어 있는 새 설치용 `default_market_scout.db` 생성
5. 로컬 프로젝트 파일을 즉시 교체하고, 두 파일을 별도로 다운로드할 수 있게 제공

Streamlit Cloud에서 버튼을 실행한 경우 배포 파일 변경은 재배포 후 사라질 수 있습니다. 생성된 `item_master.csv`와 `default_market_scout.db`를 내려받아 GitHub 프로젝트의 같은 파일과 교체하세요.

## 설치 및 교체
기존 프로젝트 파일을 ZIP의 파일로 전부 교체한 뒤 실행하세요.
기존 사용자 DB는 삭제하지 않아도 됩니다. 새 설치에서는 프로젝트에 포함된 `default_market_scout.db`가 사용자 데이터 폴더로 복사됩니다.
